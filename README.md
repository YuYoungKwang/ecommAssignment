## 실행 방법

### 1. 서버 실행

ecommAssignment\server 경로로 이동후 터미널을 열고 실행합니다.
```bash
npm install
npm run dev
```
서버는 http://localhost:3000 에서 실행됩니다.


### 2. 프론트 엔드 실행
ecommAssignment\frontend 경로로 이동후 터미널을 열고 실행합니다.
```bash
npm install
npm run dev
```
프론트엔드는 http://localhost:5173 에서 실행됩니다.

### 주의 사항
-서버가 먼저 실행되어 있어야 프론트에서 데이터를 불러올 수 있습니다.  
-서버 개발 실행은 npm run dev를 사용합니다.  
-방송 테이블은 http://localhost:5173 주소에서 확인 가능합니다.
